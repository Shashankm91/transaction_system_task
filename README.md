# Laravel Transaction Management System

## Tech Stack
- Laravel 12
- MySQL
- Blade + Bootstrap
- REST API

## Features
- Server-side pagination (Load More)
- Transaction filtering (ID & Status)
- Daily transaction summary calculation
- Manual job trigger
- Dashboard reporting
